# はじめに

## このチュートリアルについて

このチュートリアルは、TypeScript と Express を使った REST API の作り方を学ぶための実践的な教材です。実際に動作する ToDo 管理 API を段階的に構築しながら、バックエンド開発の基礎を習得できます。

## 学習のゴール

このチュートリアルを通じて、以下のスキルを身につけることができます：

- **Node.js + TypeScript + Express** で REST API を構築できるようになる
- **Prisma** でデータベース操作ができるようになる
- **テスト**の書き方を学ぶ
- **Docker化**して**Cloud Run**にデプロイできるようになる

## 前提知識

このチュートリアルを始める前に、以下の知識があることを前提としています：

- **JavaScript の基礎**
  - 変数、関数、配列、オブジェクトの基本的な操作
  - async/await の使い方
  - ES Modules (import/export) の理解
- **Git の基礎**
  - clone, commit, push などの基本的なコマンド操作

Node.js や TypeScript、Express の知識は不要です。このチュートリアルで一から学んでいきます。

## 作るもの

### ToDo 管理 API

シンプルな ToDo（タスク）管理システムの API を構築します。

**主な機能：**
- ToDo の一覧取得
- ToDo の新規作成
- ToDo の更新（完了/未完了の切り替え）
- ToDo の削除
- フィルタ（完了済み/未完了）

**技術スタック：**
- **Runtime**: Node.js v22
- **言語**: TypeScript
- **フレームワーク**: Express v5
- **バリデーション**: Zod
- **ORM**: Prisma
- **データベース**: SQLite（開発用）
- **テスト**: Vitest
- **Lint/Format**: Biome

## SPA との接続

このチュートリアルでは、完成済みの Single Page Application（SPA）を提供しています。あなたが作成した API に接続することで、実際にブラウザから ToDo を操作できます。

```
┌─────────────────────────────────────────┐
│  📝 ToDo App                            │
├─────────────────────────────────────────┤
│  ┌─────────────────────────┐  [追加]    │
│  │ 新しいタスクを入力...    │            │
│  └─────────────────────────┘            │
│                                         │
│  [全て] [未完了] [完了済み]              │
│                                         │
│  ☐ 牛乳を買う                    [削除] │
│  ☑ TypeScriptの勉強             [削除]  │
│  ☐ APIチュートリアル作成         [削除] │
│                                         │
└─────────────────────────────────────────┘
```

## チュートリアルの進め方

### 構成

このチュートリアルは全7章で構成されています：

1. **プロジェクトセットアップ** - 開発環境の構築
2. **Express 基礎** - フレームワークの基本を学ぶ
3. **REST API 設計と実装** - ToDo API をインメモリで実装
4. **Prisma でデータ永続化** - データベースと接続
5. **テスト** - API のテストを書く
6. **Docker化** - コンテナイメージを作成
7. **Cloud Run へデプロイ** - 本番環境へデプロイ

### リポジトリの使い方

このリポジトリには以下のディレクトリが含まれています：

- **`backend/`** - あなたの実装のスタート地点
- **`complete/`** - 模範解答（各章完了時点のコード）
- **`frontend/`** - 配布用の SPA
- **`docs/`** - チュートリアルドキュメント（今読んでいるもの）

### 学習の流れ

1. このリポジトリをテンプレートとして、自分のリポジトリを作成
2. `backend/` ディレクトリで実装を進める
3. 各章のドキュメントを読みながらコードを書く
4. 詰まったら `complete/` ディレクトリの模範解答を参照
5. 各章の終わりに章末チェックリストで確認

### ブランチ/タグについて

各章完了時点のコードは、ブランチまたはタグで参照できます：

```
chapter/1  → 第1章完了時点
chapter/2  → 第2章完了時点
chapter/3  → 第3章完了時点
...
chapter/7  → 第7章完了時点（完成形）
```

困ったときは、対応するブランチを checkout して参照してください。
